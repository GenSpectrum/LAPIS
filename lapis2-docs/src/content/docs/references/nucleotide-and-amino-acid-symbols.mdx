This page lists valid symbols for nucleotides and amino acids.

## Nucleotides

| Symbol | Meaning           | Ambiguous Symbol |
| ------ | ----------------- | ---------------- |
| A      | Adenine           |                  |
| C      | Cytosine          |                  |
| G      | Guanine           |                  |
| T      | Thymine           |                  |
| -      | Deletion          |                  |
| N      | failed read / any |                  |
| R      | A or G            | ✔               |
| Y      | C or T            | ✔               |
| S      | C or G            | ✔               |
| W      | A or T            | ✔               |
| K      | G or T            | ✔               |
| M      | A or C            | ✔               |
| B      | not A             | ✔               |
| D      | not C             | ✔               |
| H      | not G             | ✔               |
| V      | not T             | ✔               |
